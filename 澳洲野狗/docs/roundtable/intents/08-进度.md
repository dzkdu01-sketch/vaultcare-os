# 意图：/进度

**别名：** `/日报` `/sync`

**触发时机：** 同步每日进展

---

## 作用

结构化记录每日进展，更新状态文件，生成交接信息。

---

## 执行流程

```
1. 追问：今天完成了什么？
2. 追问：遇到什么阻塞？
3. 追问：明天计划做什么？
4. 更新 current-status.md
5. 生成进度摘要
```

---

## 进度模板

```markdown
## 今日完成
- [x] 任务 1
- [x] 任务 2

## 阻塞问题
- [ ] 问题 1（需要 XX 协助）

## 明日计划
- [ ] 任务 1
- [ ] 任务 2
```
