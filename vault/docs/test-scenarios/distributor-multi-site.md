## 四、导出（给站点上架用）

### 场景 DMS-06：按当前筛选导出商品 CSV
- 关联母表：`PIM-1.4.1`
- 频率：高频
- 步骤：
  - 在商品列表设置筛选条件
  - 点击“导出”
- 预期：下载 CSV，编码为 utf-8-sig，字段与筛选结果一致
- 当前状态：🟣 已实现-待验证（需要实际点一次并检查文件）

### 场景 DMS-07：分站点选品导出（UAE站 / TH站）
- 关联母表：`DIST-3.2.3`、`FX-4.1.1`
