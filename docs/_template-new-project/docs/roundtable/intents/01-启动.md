# 意图：/启动

**别名：** `/开始` `/init`

**触发时机：** 新对话开始

---

## 作用

自动读取项目上下文，输出当前状态摘要，快速进入工作状态。

---

## 执行流程

```
1. 读取 @docs/status/current-status.md
2. 读取 @docs/handover.md
3. 输出摘要：
   - 当前状态
   - 待修复 Bug 列表（P0/P1）
   - 待办事项
4. 询问："今天想做什么？"
```

---

## 注意事项

- **状态标识**：启动后每次回复开头带 `[人机协作-启动]`，直至用户切换其他意图
- 输出保持简洁，不超过 20 行
- 最后必须询问用户今天的目标
