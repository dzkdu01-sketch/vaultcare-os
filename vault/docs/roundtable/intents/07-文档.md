# 意图：/文档

**别名：** `/更新文档` `/doc`

**触发时机：** 代码变更后更新文档

---

## 作用

自动对比代码变更，生成文档更新建议，保持文档同步。

---

## 执行流程

```
1. 自动对比代码变更
2. 列出需要更新的文档：
   - current-status.md（API 变更）
   - 模块 spec（功能变更）
   - ops-processes.md（流程变更）
3. 生成更新建议
4. 用户确认后执行
```

---

## 文档映射关系

| 代码变更类型 | 需要更新的文档 |
|--------------|----------------|
| 新增 API | current-status.md API 列表 |
| 修改功能 | 模块 spec |
| 流程变更 | ops-processes.md |
| Bug 修复 | current-status.md Bug 列表 |

---
