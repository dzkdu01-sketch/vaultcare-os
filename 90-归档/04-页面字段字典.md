## 1. 使用说明

这份文档不追求把所有字段一次写完，而是先定义一期 8 个页面的**最小必需字段**。

字段说明统一采用：

- `字段名`：建议的系统字段键
- `页面显示名`：页面上给用户看的名称
- `类型`：文本 / 枚举 / 数字 / 布尔 / 日期时间 / 列表 / 对象
- `必填`：是否为一期必填
- `来源`：人工填写 / 系统生成 / 外部回传 / AI 建议
- `谁可编辑`：老板 / 运营审单 / 分销商 / 系统
- `说明`：字段的业务用途或特别规则
