## 一、编码规范

| 项目 | 规则 |
|------|------|
| 格式 | `VC-{4位序号}`，如 `VC-0001` |
| 生成方式 | 系统自增，运营不可编辑 |

## 二、图片规范

### 2.1 数量与类型

| 类型 | 数量 | 说明 |
|------|------|------|
| 主图 | 1 张 | 纯白背景，产品正面居中 |
| 细节图 | 2-4 张 | 不同角度、细节特写、功能展示 |
| 场景图 | 1 张（可选） | 使用场景或包装展示 |
| **总计** | **4-6 张** | |

### 2.2 技术要求

| 项目 | 要求 |
|------|------|
| 尺寸 | 1000 × 1000 px |
| 比例 | 1:1 正方形（必须） |
| 格式 | JPG 或 PNG |
| 大小 | 每张 < 500KB |

### 2.3 质量要求

必须满足：

- 主图纯白背景
- 高清无模糊
- 产品居中摆放

禁止出现：

- 水印（中文、供应商 Logo、价格标签等）

### 2.4 排序规则

不要求文件命名，按上传顺序管理：

| 位置 | 内容 |
|------|------|
| 第 1 张 | 主图（纯白背景正面） |
| 第 2 张 | 视频（如有） |
| 第 3-6 张 | 细节图 / 场景图 |

系统侧用 `position` 字段控制排序，主图自动标记 `is_primary`。

## 三、视频规范

### 3.1 技术要求

| 项目 | 要求 |
|------|------|
| 数量 | 最多 1 个 |
| 比例 | 1:1 正方形（与图片一致） |
| 分辨率 | 720p 或 1080p |
| 格式 | MP4（H.264 编码） |
| 时长 | 15-30 秒 |
| 大小 | < 20MB |

### 3.2 内容要求

必须包含：

- 产品外观展示（360度旋转或多角度）
- 核心功能演示（如振动、加热、遥控等）

禁止出现：

- 供应商水印或 Logo
- 中文字幕或中文配音

### 3.3 位置

放在图库第 1 或第 2 位。

### 3.4 无视频时的处理

- 不上传低质量视频
- 只上传高质量图片即可
- 后续补充视频时再更新

## 四、标题规范

### 4.1 基本规则

| 项目 | 规则 |
|------|------|
| 总字符数 | ≤ 55 |
| 编码部分 | 系统自动拼接，运营只填产品名部分 |
| 大小写 | Title Case（每个实词首字母大写） |

### 4.2 结构

```
VC-XXXX [尺寸(仅敏感品类)] [产品核心类型] [关键差异点]
```

| 部分 | 必填 | 说明 |
|------|------|------|
| 编码 | 是 | 系统自动拼接 |
| 尺寸 | 仅敏感品类 | 见[附录B](#附录b尺寸敏感品类清单) |
| 产品核心类型 | 是 | 品类通用词 |
| 关键差异点 | 是 | 1 个最突出的卖点或规格 |

### 4.3 禁止内容

- 年份（2024、2025 等）
- 形容词堆砌（Best / New / Hot / High Quality）
- 全大写
- 促销词（Free Shipping / Sale / Discount）

### 4.4 示例

| 标题 | 字符数 |
|------|--------|
| `VC-0276 Mini Thrusting Machine with Heating` | 45 |
| `VC-0279 12.2in Realistic Dildo with Suction Cup` | 48 |
| `VC-0385 Silicone Vibrator with APP Control` | 43 |
| `VC-0874 Linda 158cm Full Body Silicone Doll` | 44 |
| `VC-0462 Dual-Ended Electric Masturbator` | 40 |
| `VC-0102 Auto Stroker with Heating` | 33 |

### 4.5 常见错误

| 错误示例 | 问题 | 正确示例 |
|----------|------|----------|
| VC276 PORTABLE, COMPACT, AND MINI THRUSTING DILDO | 太长、全大写、形容词堆砌 | VC-0276 Mini Thrusting Machine with Heating |
| High Quality Silicone Vibrator For Women Pink Color | 无编号、太长 | VC-0385 Silicone Vibrator with APP Control |
| 2024 New Style Automatic Male Masturbator | 有年份、无编号 | VC-0102 Auto Stroker with Heating |
| vibrator | 无编号、太简单、无大写 | VC-0520 Compact Vibrator |

## 五、短描述规范

### 5.1 基本规则

| 项目 | 规则 |
|------|------|
| 总字符数 | ≤ 150 |
| 结构 | 单段式，仅卖点列表 |
| 卖点数量 | 固定 3 个 |
| 分隔符 | 圆点 `·` |
| 语言 | 英文，简洁有力 |

按以下顺序挑选 3 个最重要的卖点：

| 优先级 | 类型 | 示例 |

| 4 | 便利性 | USB Rechargeable · Waterproof IPX7 · Discreet Packaging |

```
7 Thrusting Modes · Smart Heating 37°C · Wireless Remote
```

```
Medical-grade Silicone · 10 Vibration Patterns · Waterproof IPX7

```

## 六、长描述规范

### 6.1 结构

长描述必须包含以下三个部分，按顺序排列：

1. **FEATURES**（功能特点）
2. **SPECIFICATIONS**（产品规格）
3. **PACKAGE INCLUDES**（包装清单）

### 6.2 FEATURES 规则

| 项目 | 规则 |
|------|------|
| 条数 | 固定 5 条 |
| 句式 | 每条动词开头 |
| 长度 | 每条 ≤ 80 字符 |
| 内容 | 先写体验感受，再写技术参数 |

示例：

```html
<h3>FEATURES</h3>
<ul>
    <li>Delivers powerful thrusting with 7 adjustable speed modes</li>
    <li>Heats up to 37°C for a realistic and comfortable experience</li>
    <li>Controls wirelessly with included remote up to 10m range</li>
    <li>Charges quickly via USB magnetic cable in 90 minutes</li>
    <li>Operates at whisper-quiet levels under 50dB</li>
</ul>
```

### 6.3 SPECIFICATIONS 规则

4 项必填 + 按品类选填：

| 字段 | 必填 | 说明 |
|------|------|------|
| Material | 是 | 材质 |
| Size | 是 | 格式：`11.5cm (4.5")` |
| Weight | 是 | 格式：`530g (18.7oz)` |
| Power | 是 | 供电方式（USB / Battery / Plug-in） |
| Waterproof | 否 | 防水等级（适用时填写） |
| Noise Level | 否 | 噪音（适用时填写） |
| Battery | 否 | 电池容量（适用时填写） |
| Charging Time | 否 | 充电时长（适用时填写） |

示例：

```html
<h3>SPECIFICATIONS</h3>
<ul>
    <li>Material: Medical-grade silicone + ABS</li>
    <li>Insertable Length: 11.5cm (4.5")</li>
    <li>Total Length: 22cm (8.7")</li>
    <li>Weight: 530g (18.7oz)</li>
    <li>Power: USB magnetic charging</li>
    <li>Waterproof: IPX6 splash proof</li>
    <li>Battery: 1200mAh lithium</li>
</ul>
```

### 6.4 PACKAGE INCLUDES 规则

| 项目 | 规则 |
|------|------|
| 条数 | 4-6 条 |
| 排序 | 第一项永远是主产品 |

示例：

```html
<h3>PACKAGE INCLUDES</h3>
<ul>
    <li>1x Thrusting device</li>
    <li>1x Wireless remote control</li>
    <li>1x USB charging cable</li>
    <li>1x User manual</li>
    <li>1x Velvet storage pouch</li>
</ul>
```

## 十、系统自动校验项

### 10.1 强制拦截（不通过不让提交）

| 校验项 | 校验逻辑 |
|--------|---------|
| 图片数量 4-6 张 | `len(images) >= 4 and <= 6` |
| 图片 1:1 比例 | 上传时读取宽高 `width == height` |
| 单张图片 < 500KB | 上传时校验文件大小 |
| 视频 < 20MB | 上传时校验文件大小 |
| 视频时长 15-30 秒 | 上传时读取 duration |
| 标题 ≤ 55 字符 | 输入框实时计数 |
| 短描述 ≤ 150 字符 | 输入框实时计数 |
| 长描述含三段结构 | 检测 FEATURES / SPECIFICATIONS / PACKAGE INCLUDES 关键词 |
| 售价 < 划线价 | 数值比较 |

### 10.2 黄色警告（提示但不拦截）

| 校验项 | 校验逻辑 |
|--------|---------|
| 标题 Title Case | 正则校验首字母大写 |
| 短描述含 `·` 分隔符 | 检测 `·` 字符 |

### 10.3 AI 辅助（后续迭代）

| 校验项 | 说明 |
|--------|------|
| 图片水印检测 | AI 图像识别 |
| 图片模糊检测 | AI 质量评分（`ProductMedia.quality_score`） |
