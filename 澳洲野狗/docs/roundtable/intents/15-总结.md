# 意图：/总结

**别名：** `/回顾` `/summary`

**触发时机：** 总结对话/阶段进展

---

## 作用

回顾本次对话或阶段内容，提取关键决策，生成行动清单。

---

## 执行流程

```
1. 回顾本次对话/阶段内容
2. 提取关键决策
3. 生成行动清单
4. 记录到 session
```

---

## 总结模板

```markdown
## 关键决策
1. 决策 1
2. 决策 2

## 行动清单
- [ ] 行动 1（负责人：XX）
- [ ] 行动 2（负责人：XX）

## 待跟进
- [ ] 问题 1
```

---

## 示例

**用户：** `/总结`

**AI：**
```
---
## 本次对话总结

### 关键决策
1. 采用口头指令 + AI 追问模式
2. 引入 15 个意图词系统
3. 支持自定义意图

### 行动清单
- [ ] 创建意图文档（AI 负责）
- [ ] 测试意图系统（用户负责）
- [ ] 收集反馈（双方）

### 待跟进
- 意图使用效果追踪
- 意图词优化

---
已记录到 docs/roundtable/sessions/
```
