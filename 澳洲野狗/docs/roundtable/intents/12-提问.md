# 意图：/提问

**别名：** `/问` `/ask`

**触发时机：** 技术/业务问题即时解答

---

## 作用

快速回答技术或业务问题，提供准确参考信息。

---

## 执行流程

```
1. 接收问题
2. 判断是否需要上下文
3. 输出答案 + 参考来源
4. 询问是否需要深入
```

---

## 问题分类

| 类型 | 处理方式 |
|------|----------|
| 技术问题 | 直接回答 + 代码示例 |
| 业务问题 | 查阅文档 + 回答 |
| 项目问题 | 读取相关文件 + 回答 |

---

## 示例

**用户：** `/提问 Django 如何处理跨域请求`

```
---
## 回答

Django 处理跨域请求使用 `django-cors-headers` 库：

### 安装
```bash
pip install django-cors-headers
```

### 配置
```python
# settings.py
INSTALLED_APPS = [
    'corsheaders',
    ...
]
MIDDLEWARE = [
    'corsheaders.middleware.CorsMiddleware',
    ...
]
CORS_ALLOWED_ORIGINS = [
    "http://localhost:3000",
]
```

### 参考
- 官方文档：https://django-cors-headers.readthedocs.io/
---
需要我深入讲解某个部分吗？
```
