## 圆桌游戏设计方案总结

### 1. 核心设定

| 维度 | 内容 |
|------|------|
| 题材 | 玩家掌握可在计算机中演化的 DNA 改造技术 |
| 初始资源 | 500 万美元 |
| 世界观 | 其余设定尽量贴近现实世界 |

### 1.1 开始剧情（2026-03 圆桌追加）

玩家拥有 DNA 改造技术，想**租个海岛、建个猎场**。通过改造 DNA 培育更强猎犬，**销售给中国有钱人**（高端猎犬市场）。这是 Phase 1 的入口叙事与动机。

**形式参考：** 中国护农狩猎队生态——猎犬围猎、赏金捕猎、头狗/重托等分工。可参考：[南方周末《赏金猎人：追野猪，追流量》](https://www.nfnews.com/content/r6dXl05O6E.html)、[护农狩猎队](https://www.youtube.com/hashtag/护农狩猎队)。

### 2. 玩法定位

| 维度 | 选择 |
|------|------|
| 主类型 | 模拟经营 |
| 主玩法 | 资源分配 + 环境/随机事件（无博弈对手） |
| 演化机制 | 参数式：调滑块（环境压力、突变率、世代数等）→ 系统跑结果 → 玩家选择 |
| 时间 | 混合：基本回合 + 事件打断 |
| 规模 | 地点限定中国海岛（玩家租岛建猎场），物种先从猎犬起、后期可扩展大型猫科 |

### 3. 三阶段结构

| 阶段 | 内容 |
|------|------|
| **Phase 1：准备期** | 入口：租中国海岛建猎场 → 改造 DNA 培育猎犬卖向中国富人 → 资源分配、演化参数调节、随机事件；约 10 回合 |
| **Phase 2：转折** | 条件/随机触发：意外事件使世界进入「无重武器」混乱状态 |
| **Phase 3：征服期** | 用演化成果作战力，在地图上扩张，关键节点以事件与选择推进；地图、人口、经济尽量贴近现实 |

### 4. 技术载体

- **形式**：Web 单页（HTML + JS）
- **UI**：滑块、简易地图、事件日志、操作按钮
- **实现**：无后端，状态存 localStorage

### 5. 实现策略（方案 A）

- 先做 Phase 1 完整可玩
- Phase 2 用简化条件触发
- Phase 3 先用大洲/区域级地图跑通，再细化到国家级与真实人口/经济数据

### 6. 节奏与征服期方向（2026-03 圆桌追加）

| 维度 | 决策 |
|------|------|
| **Phase 1 节奏** | 混合型：基础回合快 + 关键节点节奏放慢；大约 10 回合完成准备期 |
| **Phase 2 转折** | 保持条件/随机触发，玩家无法预知具体时点 |
| **Phase 3 征服期** | 混合型，**事件为主**：地图是区域解锁器，占领触发该区专属事件；核心体验来自事件与选择，而非纯数值领土博弈 |

### 7. 实验基地与海岛数据（2026-03 圆桌追加）

| 维度 | 决策 |
|------|------|
| **Phase 1 入口** | 玩家从候选中国海岛中**租**海岛作为猎场，再进入准备期 |
| **海岛数据要求** | 真实岛屿名称 + 真实地理位置；真实岛上地理环境（地形、气候、资源）；真实人口；真实生物与生态环境 |
| **数据粒度** | 细粒度：到具体城市/区域级（如文昌：人口 X 万，海岸线 Y 公里） |
| **数据来源** | 更权威来源（年鉴、学术文献、官方统计等） |

### 8. 物种演进路线（2026-03 圆桌追加）

| 阶段 | 物种 | 说明 |
|------|------|------|
| **Phase 1 主力** | 猎犬 | 先改造狗，培育更强猎犬卖向中国富人市场 |
| **后期扩展** | 大型猫科（如老虎） | 可改造为可团队协作的变体，供 Phase 2/3 使用 |
| **Phase 2/3 衔接** | 先不确定 | 猎犬在 Phase 3 的具体用途，后面再细化 |

## 设计共识（圆桌结论）

- 不设博弈对手，压力来自资源 scarcity、环境事件、科学不确定性
- 转折由系统条件/随机判定，玩家无法预知具体时点
- 征服期：混合型，事件为主；地图 + 事件选择，地理/人口/经济贴近现实
- Phase 1 开始剧情：租海岛建猎场，改造 DNA 培育猎犬，销售给中国有钱人；形式参考护农狩猎队生态
- Phase 1 入口：租中国海岛作为猎场；大约 10 回合，混合节奏
- 物种演进：先狗（猎犬）→ 后期可扩展大型猫科（如老虎改为团队协作）；Phase 2/3 猎犬用途后定
- 海岛数据：真实名称、位置、地理、人口、生态；细粒度；更权威来源
- 灵感示例：中华田园犬在澳洲演化成另一方向 → 与项目名「澳洲野狗」呼应
