# 意图：/飞轮报告

**版本**: 1.0 (2026-03-03)

---

## 触发词

- `/飞轮报告`
- `/飞轮`
- `/数据报告`
- `/协作数据`

---

## 作用

收集并展示人机协作体系的使用数据，用于持续改进。

---

## 流程

### 1. 执行数据收集

运行后端脚本 `backend/flywheel/collector.py`

### 2. 输出内容

AI 自动输出以下信息：

```markdown
## 📊 飞轮数据报告

**报告周期**: 第 X 周 (YYYY-MM-DD)

### 核心指标

| 指标 | 数值 |
|------|------|
| 最常用意图 | /XXX |
| 意图总使用次数 | X 次 |
| Bug 修复率 | XX% |
| 待修复 Bug | X 个 |
| 文档活跃度 | XX% |

### 意图使用情况

（表格展示各意图使用频率）

### Bug 统计

（按优先级分布）

### 文档更新

（本周更新的文档列表）

### 改进建议

（根据数据自动生成建议）
```

### 3. 追问流程

如果数据不足，AI 会提示：

```
数据收集完成。发现以下情况：

1. 会话文件数量为 0，无法统计意图使用
   → 建议：开始使用意图系统（如 /启动 /开发）

2. 本周无文档更新
   → 建议：代码变更后及时更新文档

是否需要生成详细报告文件？（是/否）
```

---

- `docs/flywheel/weekly-report-YYYY-WXX.md` - Markdown 报告
