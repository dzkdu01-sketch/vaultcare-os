# 意图：/部署

**别名：** `/上线` `/deploy`

**触发时机：** 上线发布

---

## 作用

规范化上线流程，确保所有检查项通过，降低上线风险。

---

## 执行流程

```
1. 检查上线清单：
   - 所有测试通过
   - 文档已更新
   - 无 P0/P1 Bug
2. 生成部署步骤
3. 确认后执行
```

---

## 上线清单

### 前置检查
- [ ] 所有测试通过
- [ ] 代码审查完成
- [ ] 文档已更新
- [ ] 无 P0/P1 Bug

### 部署步骤
- [ ] 备份数据库
- [ ] 拉取最新代码
- [ ] 执行迁移
- [ ] 重启服务
- [ ] 验证功能

### 回滚计划
- [ ] 回滚步骤已定义
- [ ] 备份可用
