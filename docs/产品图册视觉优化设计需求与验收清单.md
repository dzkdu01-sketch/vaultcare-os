# 产品图册视觉优化设计需求与验收清单

## 一、项目说明
本次优化对象为现有产品图册。
**优化范围仅限视觉设计层面**，不涉及内容增删、不调整产品顺序、不修改文案信息。

## 二、优化目标
在保持现有内容结构不变的前提下，提升图册的整体专业感、整洁度和对外展示质感，使其更适合作为发给客户的产品资料。

目标关键词如下：

- 简洁
- 高级
- 统一
- 清晰
- 克制

整体希望做到：
**视觉更精致，但不影响客户快速看产品，且产品本身始终为页面主角。**

## 三、设计原则
1. **不改内容**
   - 不新增、不删减、不重写文案
   - 不调整产品信息逻辑
   - 不改变现有产品展示内容

2. **只做视觉优化**
   - 重点调整版式、留白、层级、色彩、标签样式、章节感和整体统一性

3. **以产品为主**
   - 所有视觉优化都应服务于产品展示
   - 不做过重装饰，不喧宾夺主

## 四、设计需求清单

### 1. 页面整体版式优化
- 增加页面四周外边距，提升整体留白感
- 统一各页版心宽度和内容排布节奏
- 拉开产品卡片之间的横向、纵向间距，减少拥挤感
- 保证整本图册版式松紧一致，避免有的页面过密、有的页面过散

### 2. 分类与章节区域优化
- 强化分类标题区域的完整性和层次感
- 增加分类标题上下留白，避免分类区像简单分隔线
- 章节切换处形成清晰的视觉停顿，增强浏览节奏
- 分类样式需统一，确保整本图册有稳定的章节识别感

### 3. 产品卡片系统统一
- 统一产品卡的宽度、高度、间距和内部结构
- 统一图片在卡片中的展示比例和对齐方式
- 尽量统一不同产品图的视觉重心、摆放位置和底部落点
- 保证同一页面内产品展示整齐稳定，避免视觉跳动

### 4. 信息层级优化
- 优化产品名、编号、价格等信息的层级关系
- 统一文字字号、字重、行距、段距和排列逻辑
- 提升信息可读性，减少页面信息“同权化”带来的平感
- 确保客户浏览时能快速识别产品主体与辅助信息

### 5. 色彩与视觉质感优化
- 页面背景建议从纯白优化为更柔和的浅灰白体系
- 文字颜色建议从纯黑调整为深灰体系，提升精致感
- 分隔线建议更细、更浅，减少页面表格感
- 如需提升结构感，可使用极浅边框或非常轻的阴影，不宜过重

### 6. 标签与价格区样式优化
- 现有黑色标签/价格块视觉权重偏高，建议减弱处理
- 保留识别功能，但降低对页面主视觉的干扰
- 样式建议更轻量、更克制，避免过强电商促销感
- 整体应与图册高级、整洁的气质保持一致

## 五、整体视觉方向
请整体控制在以下风格方向内：

**极简、干净、统一、克制、精致**

不希望呈现为：
- 电商促销页风格
- 杂志化过强风格
- 装饰元素过多的版式风格

希望呈现为：
- 更像专业整理过的品牌产品资料
- 更适合直接发给客户查看
- 浏览轻松，视觉安静，重点明确

## 六、最终效果要求
优化后的图册应达到以下感受：

- 页面更透气，浏览不累
- 分类更清楚，章节更完整
- 产品陈列更整齐，整体更统一
- 视觉更高级，但不影响产品识别
- 客户第一眼会感觉资料更专业、更有品质感

## 七、一句话总结
**在不改任何内容的前提下，通过留白、对齐、统一、减重和章节感优化，提升整本产品图册的专业度与高级感，并确保产品始终是视觉主角。**

# 设计验收清单

## 一、范围验收
- [ ] 未修改原有产品内容
- [ ] 未调整原有产品顺序
- [ ] 未修改原有文案信息
- [ ] 本次改动仅限视觉设计层面

## 二、整体版式验收
- [ ] 页面四周外边距明显比原稿更宽松
- [ ] 各页版心宽度统一，没有忽大忽小的问题
- [ ] 产品卡之间横向、纵向间距统一
- [ ] 页面整体松紧一致，没有局部过密或过空现象
- [ ] 长图连续浏览时节奏更稳定，不显拥挤

## 三、分类与章节验收
- [ ] 分类标题区域比原稿更完整，不再像简单分隔线
- [ ] 分类标题上下留白明显增强
- [ ] 章节切换处有清晰的视觉停顿
- [ ] 各分类样式统一，整本图册章节识别感一致

## 四、产品卡片验收
- [ ] 产品卡宽度、高度、间距保持统一
- [ ] 产品图片展示比例统一，没有忽大忽小的问题
- [ ] 产品图在卡片中的对齐方式统一
- [ ] 不同产品图的视觉重心更稳定
- [ ] 产品图底部落点更整齐
- [ ] 同一页面产品排列观感整齐，没有明显跳动

## 五、信息层级验收
- [ ] 产品名、编号、价格的层级关系比原稿更清晰
- [ ] 字号、字重、行距、段距有统一规则
- [ ] 客户扫读时能快速分辨主信息与辅助信息
- [ ] 页面不再出现信息全部同权、阅读发平的问题

## 六、色彩与质感验收
- [ ] 页面背景观感更柔和，不再显得生硬
- [ ] 文字颜色整体更细腻，不再过于死黑
- [ ] 分隔线更轻、更细，不再有明显表格感
- [ ] 如使用边框或阴影，强度足够克制，不影响高级感
- [ ] 整体视觉更加干净统一，没有额外装饰噪音

## 七、标签与价格区验收
- [ ] 标签/价格区仍然清晰可识别
- [ ] 标签/价格区的视觉重量较原稿有所减弱
- [ ] 标签/价格区不会抢夺产品图的主视觉地位
- [ ] 整体风格不呈现强促销感或电商海报感

## 八、最终观感验收
- [ ] 产品仍然是页面主角
- [ ] 整本图册比原稿更干净、更高级、更专业
- [ ] 浏览体验更轻松，连续查看不容易疲劳
- [ ] 分类更明确，页面更有秩序
- [ ] 图册整体更适合作为发给客户的正式资料

## 九、最终验收标准
若最终设计稿同时满足以下条件，则视为通过：

- 不改内容，只优化视觉
- 页面更统一、更透气、更整洁
- 分类更有章节感
- 产品卡更稳定，图片更整齐
- 标签与价格区被有效减重
- 产品仍是全册第一视觉主体
- 整体观感明显优于原稿，并可直接对外发送

## 十、实现与验收方式（工程）

- **生成位置**：客户下载的 PNG 长图由后端 `backend/src/services/catalogPng.ts` 渲染，与本文档「二～六节」方向对齐（留白、章节、卡片统一、层级、柔和底色与细线、SKU/价格区减重等）；**不改变**商品列表顺序、文案与字段内容，仅调整版式与样式。
- **如何验收**：在「客户图册」页下载 For Him / For Her 的 PNG，对照**第七节～第九节**清单逐项人工勾选；若需与原稿对比，可使用同批商品在改动前的导出或 Git 历史中的实现对比。
